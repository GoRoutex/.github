## 🚀 About Go Routex

**Go Routex** is a microservices-based platform designed to power modern transportation systems.

The platform enables bus companies to:

- Manage routes and trip schedules
- Handle ticket booking and seat reservations
- Manage driver profiles
- Handle authentication and user management
- Integrate notification systems (Email / SMS)
- Operate scalable backend infrastructure

The project focuses on **high scalability, modular architecture, and cloud-native deployment**.

---

Each service is **independently deployable** and communicates through REST APIs.

## 🏗 Core Repositories

Main repositories in this organization:

| Repository | Description |
|-------------|-------------|
| `go-routex-user-service` | User authentication and identity management |
| `go-routex-route-service` | Route scheduling and trip planning |
| `go-routex-driver-service` | Driver profile and driver management |
| `go-routex-booking-service` | Ticket booking and seat reservation |
| `go-routex-api-gateway` | API Gateway for routing external requests |
| `go-routex-notification-service` | Email and SMS notification system |
| `go-routex-deployment-config` | Kubernetes and deployment configurations |

## 🛠 Technology Stack

Go Routex is built using modern backend technologies.

### Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA

### Database
- PostgreSQL

### Infrastructure
- Docker
- Kubernetes
- GitHub Actions (CI/CD)

### API
- RESTful APIs
- OpenAPI / Swagger

---

## ⚙️ CI/CD

Each microservice repository includes automated pipelines:

- Build and test with **GitHub Actions**
- Containerize using **Docker**
- Push images to **GitHub Container Registry**
- Deploy using **Kubernetes**
